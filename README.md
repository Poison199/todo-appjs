# To-Do List App

A fully functional, interactive **To-Do List Application** built with **HTML, CSS, and JavaScript**. Manage your daily tasks with ease and track your progress visually!

---

## Features

- **Add, edit, and delete tasks** dynamically
- **Mark tasks as completed** with a checkbox
- **Progress tracking** with a visual progress bar and counter
- **Celebration animation**: Confetti animation when all tasks are completed
- **Local storage support**: Tasks are saved in the browser and persist on reload
- **Responsive design** for desktops and mobile devices


---

## Technologies Used

- **HTML5** – Structure and layout of the app
- **CSS3** – Styling, animations, and responsive design
- **JavaScript** – Dynamic functionality, local storage, progress tracking
- **Font Awesome** – Icons for buttons (edit, delete, add task)
- **tsparticles Confetti** – Confetti animation for completed tasks
